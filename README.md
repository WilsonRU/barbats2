# BarbaTS Project

- **NodeJS** (^21.x)
- **Fastify**
- **Prisma**
- **Biome**
- **Typescript**

## Setup

Install the dependencies:

```bash
npm install
```

## Get Started

1. Start the dev server:

```bash
npm run dev
```

2. Knex

```bash
npm run knex <command>
```

3. Biome Format

```bash
npm run format
```

4. Biome Lint

```bash
npm run lint
```
